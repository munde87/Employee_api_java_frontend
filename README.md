# Employee Management API

Spring Boot + MongoDB based Employee Management REST API.

## Features

- Create Employee
- Get Employee
- Update Employee
- Delete Employee

## Tech Stack

- Java
- Spring Boot
- MongoDB
- Maven
- Postman
