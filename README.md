# Employee Management API

A Spring Boot REST API for managing employee records with MongoDB integration.

## Tech Stack

* Java
* Spring Boot
* MongoDB
* Maven
* Postman

## Features

* Create Employee
* Get Employee
* Update Employee
* Delete Employee
